Включение шрифта непосредственно в `.apk` (локальные шрифты) — это самый надежный способ. В отличие от загружаемых шрифтов, локальные будут работать **без интернета** и на устройствах **без Google Services**.

Вот пошаговая инструкция для Jetpack Compose:

---

## 1. Подготовка файлов

1. Перейдите в папку вашего проекта: `app/src/main/res/`.
2. Создайте там папку **`font`** (если её нет).
3. Скопируйте туда файлы шрифтов (форматы `.ttf` или `.otf`).
* **Важно:** Имена файлов должны быть в нижнем регистре, без пробелов, начинаться с буквы и содержать только латиницу, цифры или подчеркивания (например, `my_font_bold.ttf`).

---

## 2. Описание шрифта в коде (FontFamily)

В Jetpack Compose вы объединяете разные начертания (Bold, Italic, Regular) одного шрифта в объект `FontFamily`. Обычно это делают в файле `Type.kt` (в пакете `ui.theme`).

```kotlin
import androidx.compose.ui.text.font.Font
import androidx.compose.ui.text.font.FontFamily
import androidx.compose.ui.text.font.FontWeight
import androidx.compose.ui.text.font.FontStyle
import com.example.myapp.R // Замените на ваш пакет

val MyCustomFontFamily = FontFamily(
    Font(R.font.my_font_regular, FontWeight.Normal),
    Font(R.font.my_font_bold, FontWeight.Bold),
    Font(R.font.my_font_italic, FontWeight.Normal, FontStyle.Italic)
)

```

---

## 3. Использование в Typography

Чтобы не прописывать шрифт вручную для каждого текста, лучше настроить общую типографику приложения:

```kotlin
import androidx.compose.material3.Typography
import androidx.compose.ui.text.TextStyle
import androidx.compose.ui.unit.sp

val Typography = Typography(
    bodyLarge = TextStyle(
        fontFamily = MyCustomFontFamily,
        fontWeight = FontWeight.Normal,
        fontSize = 16.sp,
        lineHeight = 24.sp
    ),
    headlineMedium = TextStyle(
        fontFamily = MyCustomFontFamily,
        fontWeight = FontWeight.Bold,
        fontSize = 28.sp
    )
)

```

---

## 4. Применение в интерфейсе

Теперь вы можете использовать настроенные стили или применять шрифт точечно.

### Способ А: Через стили (рекомендуется)

Если вы передали `Typography` в ваш `MaterialTheme`, просто используйте:

```kotlin
Text(
    text = "Привет, это мой шрифт!",
    style = MaterialTheme.typography.bodyLarge
)

```

### Способ Б: Напрямую в компоненте

```kotlin
Text(
    text = "Текст с конкретным шрифтом",
    fontFamily = MyCustomFontFamily,
    fontWeight = FontWeight.Bold
)

```

---

## Стоит обратить внимание на пару моментов:

* **Размер APK:** Каждый добавленный шрифт увеличивает размер приложения. Если шрифтов много, лучше использовать формат `.woff2` (Android поддерживает его начиная с API 26) или подмножества (subsets) шрифтов, содержащие только нужные символы (например, только кириллицу).
* **Variable Fonts:** Если вы используете вариативные шрифты (один файл для всех весов), в Compose для этого используется `Font(R.font.variable_font, variationSettings = ...)`. Это современный стандарт, который сильно экономит место.
